# Release checklist

- [ ] Lint, typecheck, tests, build, and migrations pass on supported Node.js versions.
- [ ] Custom scanner and gitleaks report no findings.
- [ ] No `.env`, database, import output, screenshot, log, backup, or deployment hostname is tracked.
- [ ] Seed/demo content is fictional or absent.
- [ ] Clean clone validation passes.
- [ ] Protected `main` requires CI and secret scan checks.
